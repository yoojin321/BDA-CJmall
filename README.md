# BDA 공모전 - CJmall 장바구니 데이터 분석

**작성자**: 이태경, 이유진

## 프로젝트 개요

본 프로젝트는 BDA(빅데이터 분석) 아이디어 공모전을 위한 프로젝트로, CJmall의 11번가 및 네이버 외부몰 장바구니 데이터를 분석하여 랜덤박스 상품 기획을 위한 인사이트를 도출합니다.

CJ제일제당은 매출이 지속적으로 상승하는 모습을 보여주었으나 타 경쟁사와 비교하였을 때 유통채널의 부족과 전반적으로 높은 가격대라는 약점과 최근 경기침체와 맞물려 소비자에게 외면을 받고 있습니다. 이러한 상황에서 간편식에 지출이 많은 1인가구의 비중이 증가함에 따라 상황에 맞게 20~30대의 1인가구를 주요 타겟으로 설정한 마케팅 전략을 고안할 필요가 있습니다.

따라서 20~30대의 편리한 식료품 구매와 동시에 오락적 요소를 더하여 해당 세대의 관심을 사로잡을 전략을 고민하였고 랜덤박스 이벤트인 **'CJ박스'**를 고안하였습니다.

## 프로젝트 목적

- 11번가 및 네이버 외부몰의 장바구니 데이터를 분석하여 고객 구매 패턴 파악
- 랜덤박스 상품 기획을 위한 최적 가격대 및 구성 상품 도출
- 데이터 기반 의사결정을 통한 마케팅 전략 수립
- **최종 목표**: 매출액 증대와 판매량 증가를 통한 CJ제일제당의 경쟁력 강화

## 분석 배경 및 필요성

### CJ제일제당의 현재 상황

#### 매출 현황
- 2022년도 연간 실적: 전년 대비 19.3% 증가한 18조 7,794억 원
- 식품사업부문: 비비고 브랜드 중심의 K-푸드 해외 판매 호조로 11조 1,042억 원 (전년 대비 +16.1%)

#### SWOT 분석

**Strength (강점)**
- 글로벌 식품부문 고성장 추세
- 'K-푸드 글로벌 영토 확장' 전략으로 미국, 유럽, 호주, 태국, 말레이시아 등 진출
- 국내와 해외 식품 사업 부문 합산 매출액 전년 대비 6% 증가 (2조 7,596억 원)

**Weakness (약점)**
- 경쟁사 대비 물류와 유통채널 부족
- 제품 가격이 전반적으로 경쟁사보다 비싼 편
- 경기침체로 인한 소비 감소와 높은 가격으로 인한 소비자 외면

**Opportunity (기회)**
- 1인 가구 증가와 물가 상승으로 간편식 요리 트렌드 확산
- 코로나 이후 HMR 긍정적 평가 71.9%
- HMR 활용 식사: 1인당 연 평균 225.5끼 → 236.5끼 (11끼 증가)

**Threat (위협)**
- 쿠팡, 롯데마트와의 납품단가 이견으로 인한 거래 중단 갈등

### STP 전략

#### 시장 세분화 (Segmentation)
- **인구통계학적 기준**: 0~10대, 20~30대, 40대 이상
- **행동적 특성 기준**: 가족단위 공동 구매 vs 1인 단위 구매

#### 표적시장 선정 (Targeting)
- **선정 시장**: 1인 단위로 식료품을 구매하는 20~30대
- **전략**: 차별적 마케팅 (해당 표적시장에 특화된 제공물 개발)

#### 포지셔닝
- CJ제일제당의 식료품은 다양한 제품군을 다루며 브랜드 인지도와 선호도가 높음
- 1인 가구 증가하는 2030 세대를 타겟으로 간편식 수요 활용
- 편리한 식료품 구매 + 오락적 요소를 통한 관심 유도

## 분석 내용

### 1. 데이터 전처리
- 결측치 처리
- 날짜 데이터 타입 변환 (주문일, 출고예정일, 운송장등록일자)
- 파생 변수 생성 (운송장등록소요일, 출고소요일)

### 2. 매출액/판매량 분석
- 상품유형별 매출액 집계
- 상품명별 판매량 및 매출액 분석
- 인기 상품 도출
- **분석 목적**: 스타, 캐시 카우, 물음표, 도그 군 파악을 통한 상품 전략 수립

### 3. 배송 분석
- 배송 유형별 분석
- 배송 소요일 분석

### 4. CJ박스 기획 분석

#### 4.1 CJ박스 가격 설정
- **가설**: 특정 구매금액의 빈도가 높을수록 고객이 부담없이 상품을 구매할 가능성이 높다
- **분석 결과**:
  - 대부분의 주문별 금액이 2만원 초중반부터 3만원 중후반까지 분포
  - 중위값: 3만원 초반
  - 2만원 후반대 주문 빈도가 낮음 (3만원 이상 구매 시 추가 혜택 제공으로 인한 현상)
- **최종 가격**: **2만원 중후반** (햇반 등 복합상품과 함께 판매하여 3만원 이상 구매 유도)
  - 2만원: 제품 종류 제한으로 랜덤성 퇴색
  - 4만원 이상: 고객 부담 증가
  - 2만원 중후반: 최적의 균형점 + 추가 구매 유도 가능

#### 4.2 CJ박스 내용물 분석
- 마감구분별(냉장, 냉동, 김치 등) 인기 상품 분석
- 상품군별 선호도 분석
- **컨셉별 구성**:
  - **끼니용**: 컵반, 치킨마요덮밥, 비비고 단호박죽, 스팸김치주먹밥, 바지락칼국수
  - **밥반찬용**: 비비고 미역국/육개장, 고메 함박스테이크/돈카츠, 스팸, 더건강한 그릴후랑크
  - **간식용**: 비비고 왕교자, 고메 소바바치킨/스윗치즈피자/핫도그

#### 4.3 CJ박스 판매일 분석
- **가설**: 
  - 금월의 특정 시기의 총 매출액이 높다면 해당 시기는 고객의 소비기간이다
  - 금주의 특정 시기의 총 매출액이 높다면 해당 시기는 고객의 소비기간이다
- **분석 결과**:
  - 총매출액 상위 6일: 29일(월), 17일(수), 30일(화), 22일(월), 23일(화), 31일(수)
  - 공통점: 5월 중후반 + 해당 주차의 초중반
  - 배경: 직장인 월급일(25일 전후) + 온라인 쇼핑몰 고객 몰림 시간대(월요일 오전 11시, 오후 2시)
- **판매 기간**: **'CJ데이'** - 매월 마지막주 월요일부터 수요일까지

#### 4.4 상품군별 선호도 분석
- **가설**: 주문량과 매출액이 많을수록 선호도가 높은 상품이다
- **분석 결과**:
  - **선호도 높은 상품군 (스타/캐시 카우)**: 스팸류, 국/탕/찌개류, 만두류, 컵반류
  - **선호도 낮은 상품군 (도그)**: 죽류, the건강한(닭가슴살/소시지류), 칼국수/잡채류
- **전략**: 인기상품과 비인기상품을 적절히 혼합하여 도그군 매출 증가 및 신제품 홍보

  - SNS 챌린지로 확산 가능성
- **콜라보 마케팅**: 인기 캐릭터나 연예인과의 콜라보를 통한 파급력 확대
- **신규 개념**: 'CJ데이'와 'CJ박스'라는 신규 프로모션으로 소비자 주목 유도

### 2. 타겟 세대 브랜드 이미지 상승

### 4. 경쟁력 강화
- **차별화 전략**: 기존 묶음상품과 차별화된 랜덤박스 전략으로 경쟁 우위 확보
